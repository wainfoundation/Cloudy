Frequently Asked Questions

Do I need a Pi Network account?

Yes, Cloudy uses Pi Network for secure authentication. Download the Pi Network app to create an account if you don't have one.

Is my data safe with Cloudy?

Absolutely. We use Pi Network's secure authentication and prioritize your privacy with robust data protection.

Can I sign up without Pi Browser?

No, the Pi Browser is required for Pi Network authentication to ensure a secure sign-up process.
